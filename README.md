# Maternidade

Landing **Auxílio-Maternidade** — Coimbra & Ruas (`maternidade.coimbraeruas.com.br`). Stack: React, Vite, TypeScript, Tailwind.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Variáveis de ambiente

Copie `.env.example` para `.env` e preencha `VITE_GTAG_ID` e conversões conforme o Google Ads / GA4.
