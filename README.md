# Maternidade

Landing **Auxílio-Maternidade** — Coimbra & Ruas (`maternidade.coimbraeruas.com.br`). Stack: React, Vite, TypeScript, Tailwind.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Variáveis de ambiente

A tag Google está em `index.html`. No `.env`, opcionalmente configure `VITE_GOOGLE_ADS_CONVERSION_SEND_TO` e eventos extras conforme o Google Ads / GA4.
